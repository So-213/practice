# psql メタコマンド・SQL コマンド 早見表

## Docker 上の DB にアクセスする

```bash
docker exec -it db-learning-postgres psql -U postgres -d db_learning
```

| オプション | 説明 |
|-----------|------|
| `-it` | 対話モード（TTY 割り当て） |
| `db-learning-postgres` | コンテナ名 |
| `-U postgres` | ユーザー名 |
| `-d db_learning` | データベース名 |

ホストの psql から接続する場合（ポート 5432 がマッピングされている前提）:

```bash
psql -h localhost -U postgres -d db_learning
```

## psql メタコマンド（バックスラッシュコマンド）

| コマンド | 説明 |
|---------|------|
| `\?` | メタコマンド一覧を表示 |
| `\q` | psql を終了 |
| `\c [DB名]` | データベースに接続 |
| `\l` | データベース一覧を表示 |
| `\dt` | テーブル一覧を表示 |
| `\dt+` | テーブル一覧（詳細付き） |
| `\d [テーブル名]` | テーブル定義を表示 |
| `\d+ [テーブル名]` | テーブル定義（詳細付き） |
| `\di` | インデックス一覧 |
| `\dv` | ビュー一覧 |
| `\df` | 関数一覧 |
| `\dn` | スキーマ一覧 |
| `\du` | ロール（ユーザー）一覧 |
| `\g` | 直前のクエリを再実行 |
| `\gx` | 直前のクエリを再実行（縦表示） |
| `\x` | 拡張表示の ON/OFF 切り替え |
| `\timing` | 実行時間表示の ON/OFF |
| `\e` | エディタでクエリを編集 |
| `\i [ファイル]` | SQL ファイルを実行 |
| `\o [ファイル]` | 出力をファイルにリダイレクト |
| `\w [ファイル]` | 現在のクエリバッファをファイルに保存 |
| `\r` | クエリバッファをクリア |
| `\conninfo` | 現在の接続情報を表示 |
| `\h [コマンド]` | SQL コマンドのヘルプを表示 |

## よく使う SQL コマンド

### DDL（データ定義）

| コマンド | 説明 |
|---------|------|
| `CREATE TABLE` | テーブル作成 |
| `ALTER TABLE` | テーブル定義の変更 |
| `DROP TABLE` | テーブル削除 |
| `CREATE INDEX` | インデックス作成 |
| `CREATE VIEW` | ビュー作成 |

### DML（データ操作）

| コマンド | 説明 |
|---------|------|
| `SELECT` | データ取得 |
| `INSERT` | データ挿入 |
| `UPDATE` | データ更新 |
| `DELETE` | データ削除 |

### トランザクション

| コマンド | 説明 |
|---------|------|
| `BEGIN` | トランザクション開始 |
| `COMMIT` | トランザクション確定 |
| `ROLLBACK` | トランザクション取り消し |

### その他

| コマンド | 説明 |
|---------|------|
| `EXPLAIN` | 実行計画を表示 |
| `EXPLAIN ANALYZE` | 実行計画＋実際の実行 |
| `VACUUM` | 不要領域の回収 |
| `\watch [秒数]` | クエリを指定秒数ごとに再実行 |

## よく使う SELECT パターン

```sql
-- 全件取得
SELECT * FROM テーブル名;

-- 条件付き
SELECT * FROM テーブル名 WHERE 条件;

-- 件数制限
SELECT * FROM テーブル名 LIMIT 10;

-- 並び替え
SELECT * FROM テーブル名 ORDER BY カラム名 DESC;

-- 件数カウント
SELECT COUNT(*) FROM テーブル名;
```
